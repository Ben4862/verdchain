Prove your evidence
is real.

VerdChain creates a cryptographic fingerprint of your files the moment they are captured — permanently recorded on a public blockchain, verifiable by anyone.

The problem

Digital files can be altered without a trace

A video, photo, or document can be modified invisibly. Deepfakes are already appearing in courtrooms.

The solution

An immutable fingerprint, recorded at the source

VerdChain anchors a mathematical signature of the original file to the blockchain. Any modification is instantly detectable.

01

Capture

The moment you take a photo or video, VerdChain computes a SHA-256 hash — a unique mathematical signature of every byte in the file.

02

Anchor

That hash is written to the Polygon blockchain — a public, decentralized ledger. The record is permanent, tamper-proof, and timestamped.

03

Verify

Anyone can drop a file on this portal. If the fingerprint matches: the file is identical to the original. If not: it has been altered.

Your file never leaves your device. Only the 64-character hash is transmitted. VerdChain stores no content — only the cryptographic proof of its existence.

Install on iPhone.

Set up the VerdChain Shortcut to anchor every photo to the blockchain in one tap.

1

Open Shortcuts

Pre-installed on every iPhone. Tap + to create a new Shortcut. Name it VerdChain Capture.

2

Take Photo

Search Take Photo → add the action → Camera: Back.

3

Send to VerdChain

Search Get Contents of URL and configure:

URL : https://verdchain-api.onrender.com/api/anchor-file Method : POST Body : File → Photo
4

Get the verify URL

Add Get Dictionary → then Get Dictionary Value → key: verifyUrl.

5

Open the result

Add Open URLs → select the Dictionary Value. The verification portal opens automatically after each capture.

6

Add to home screen

Tap the Shortcut name → Add to Home Screen → use ⛓ as the icon.
